Comparativo de Sites - Atelier Ortopedia

Roberto, preparei esse comparativo para te mostrar as duas opções que podemos seguir para criar o site da Atelier Ortopedia. Ambas têm o mesmo valor fixo de R$ 500,00 para criação do site inicial com todas as informações da clínica.

Característica GitHub (Hospedagem Gratuita) Shopify (Plataforma Profissional)
Valor da Criação do Site R$ 500,00 (valor único) R$ 500,00 (valor único)
Hospedagem Mensal Gratuita A partir de U$ 24/mês (aprox. R$ 120/mês)
Domínio Personalizado Sim, com redirecionamento para GitHub Sim, direto na Shopify
Facilidade para Atualizar Conteúdo Requer conhecimento técnico ou minha ajuda Interface simples e intuitiva
Postagem de Blog / Notícias Não disponível Sim, nativo na Shopify
Coleta de E-mails de Clientes Não (requer configuração manual) Sim, via aplicativos integrados
Relatórios e Controle de Visitas Limitado (via Google Analytics) Detalhado e nativo
Segurança e Backups Boa, depende do GitHub Excelente, com suporte e backups
Responsividade (celular/tablet) Sim Sim
Indicação Ideal Para algo simples, sem alterações frequentes Para quem quer crescer online

Observações Importantes:

  • As duas opções têm o mesmo custo inicial: R$ 500,00.
  • Se quiser algo simples e sem mensalidade, o GitHub já atende bem.
  • Se quiser mais controle, postar conteúdos, captar e-mails e evoluir com marketing, a Shopify oferece mais possibilidades.
  • O domínio personalizado (ex: www.atelierortopedia.com.br) será comprado separadamente, com valor entre R$ 40 a R$ 70 por ano.
